## install ##

From rubygems:  
~~~
    [sudo] gem install num4difftest
~~~

or from the git repository on github:  
~~~
    git clone https://github.com/siranovel/num4difftest.git  
    cd num4difftest  
    gem build *.gemspec
    gem install *.gem
~~~
